This project contains two simple Snake game examples:

- `python/snake.py` — Python version using the standard `turtle` module.
- `web/index.html`, `web/styles.css`, `web/script.js` — Browser version using HTML5 canvas and JavaScript.

## Run the Python game

1. Open a terminal in `SnakeGame/python`
2. Run:

```bash
python snake.py
```

## Run the web game

1. Open `SnakeGame/web/index.html` in a browser.
2. Use the arrow keys to move the snake.

Enjoy!
